Petstore API Test

Bu projede Swagger Petstore API kullanılarak REST-Assured ile POST /pet endpoint’i test edilmiştir.

Request body POJO ile oluşturulmuştur

Testler given / when / then yapısında yazılmıştır

Response’ta id alanının boş olmadığı, name ve status alanlarının doğru döndüğü doğrulanmıştır

Kullanılan Teknolojiler: Java, REST-Assured, TestNG, Maven
